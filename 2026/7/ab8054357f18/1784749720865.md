# Yalpana — Branding & Logo Guide

## Overview

This document catalogs Yalpana's visual identity assets as found in the project archive: logo files, favicons, social preview image, and the core design system (colors + typography) already in use across the Blade views and admin dashboard.

## Brand Colors

| Role          | Hex       | Usage                                              |
|---------------|-----------|-----------------------------------------------------|
| Navy (Primary)| `#0D1B2A` | Headers, primary text, dark UI surfaces             |
| Teal (Accent) | `#0BAB8E` | CTAs, links, highlights, active states              |
| Cream         | `#F7F3ED` | Page backgrounds, light surfaces, logo lockup backdrop |
| Deep Teal (logo mark) | `#01475B` | Primary logo mark background/ink            |
| Bright Mint (logo accent) | `#2FCDC0` | Logo accent detail                      |

> The deep teal/mint pairing found inside the logo files (`#01475B` / `#2FCDC0`) sits close to the site's Navy/Teal pair but is slightly distinct — treat it as the "logo-specific" palette rather than swapping it 1:1 with the UI Navy/Teal tokens.

## Logo Files (`images/`)

| File            | Size (px)   | Format | Notes                                                                 |
|-----------------|-------------|--------|------------------------------------------------------------------------|
| `logo.png`      | 1536 × 1024 | PNG (transparent) | Primary full-color logo mark, deep-teal ink — use on light/cream backgrounds |
| `logo.webp`     | 1536 × 1024 | WebP (transparent) | Same primary mark as `logo.png`, optimized for web (94 KB vs. 371 KB) — preferred for page `<head>`/performance-sensitive placements |
| `logo2.png`     | 1408 × 768  | PNG (cream background `#F1EBDF`) | Logo lockup on solid cream card/background — use where transparency isn't needed (e.g. emails, printed materials) |
| `logo22.png`    | 1408 × 768  | PNG (transparent) | Alternate crop/version of the mark on deep-teal + mint palette |
| `logo-white.png`| 1536 × 1024 | PNG (transparent, pure white) | Reversed/white version — **use only on dark (navy) backgrounds**; invisible on light/cream backgrounds |

### Usage Guidance
- **Light backgrounds (cream `#F7F3ED`, white):** `logo.png` / `logo.webp`
- **Dark backgrounds (navy `#0D1B2A`):** `logo-white.png`
- **Solid cream card contexts (email headers, printed docs):** `logo2.png`
- **Web performance-critical spots** (navbar, footer): prefer `logo.webp` over `logo.png`
- Do not recolor the mark outside the navy/teal/white set above; do not place `logo-white.png` on light backgrounds or the standard color mark on dark navy backgrounds — contrast fails in both cases.

## Gaps / Things Not in This Archive

- No standalone **SVG** or vector source (e.g. `.ai`, `.eps`, `.fig`) for the logo — only raster PNG/WebP exports. If you need the mark at arbitrary scale (large print, merch), you'll want to recreate it as a vector.
- No documented **minimum clear space** or **minimum size** rules for the logo.
- No secondary/monochrome (single-color navy-on-transparent) version — only full-color and full-white.
